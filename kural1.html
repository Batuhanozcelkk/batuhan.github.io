Batuhan'ın Spesyalleri

\(\displaystyle \int x^2 e^x \,dx\)

Çözüm: Parçalı integral.
\(u = x^2,
\ du = 2x\,dx,\ dv = e^x\,dx \Rightarrow v = e^x\)

\(\int x^2 e^x dx = x^2 e^x - 2 \int x e^x dx = x^2 e^x - 2(e^x(x - 1)) + C = e^x(x^2 - 2x + 2) + C\)

\(\displaystyle \int \frac{x}{(x^2 + 1)^2} \,dx\)

Yer değiştirme: \(u = x^2 + 1,\ du = 2x\,dx \Rightarrow \frac{1}{2} \int u^{-2}\,du\)

Sonuç: \(-\frac{1}{2(x^2 + 1)} + C\)

\(\displaystyle \int \sin^3(x)\cos(x)\,dx\)

Yer değiştirme: \(u = \sin(x), du = \cos(x)\,dx\)

\(\int u^3 du = \frac{u^4}{4} + C = \frac{\sin^4(x)}{4} + C\)

\(\displaystyle \int \frac{dx}{x\ln(x)}\)

Yer değiştirme: \(u = \ln(x), du = \frac{1}{x}dx\)

\(\int \frac{1}{u}du = \ln|u| + C = \ln|\ln(x)| + C\)

\(\displaystyle \int x^3 \sqrt{x^2 + 1}\,dx\)

Yer değiştirme: \(u = x^2 + 1,\ du = 2x\,dx \Rightarrow \frac{1}{2} \int (u-1)\sqrt{u}\,du\)

Sonuç: \(\frac{1}{2}\left(\frac{2}{5}u^{5/2} - \frac{2}{3}u^{3/2}\right) + C = \frac{1}{5}(x^2+1)^{5/2} - \frac{1}{3}(x^2+1)^{3/2} + C\)
